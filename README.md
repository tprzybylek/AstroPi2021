## Directory structure

```
preprocessing
├── learn               stacked crops separated for model training and testing
├── crops               list of rectangles to be cropped from georeferenced images
├── glc                 GeoTIFF images with landcover classification 
├── stacked             cropped georeferenced images
├── georeferenced       raw georeferenced images
└── source              raw images from past experiment
results
├── pruscode            raw data from experiment
└── predictions         output data
```
